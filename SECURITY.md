# Security Policy

## Supported versions

| Version | Supported |
| ------- | --------- |
| latest  | Yes       |

## Reporting a vulnerability

If you discover a security vulnerability, please report it privately via [GitHub Security Advisory](https://github.com/JuroOravec/safe-ai-factory/security/advisories/new). Do not open a public issue.

You can expect an initial response within 48 hours and a resolution timeline within 7 days.
